# 🔧 FIXZO - Home Service Booking Platform

FIXZO is a modern and responsive **Home Service Booking Platform** that allows users to book trusted professionals for various household services such as Electrical, Plumbing, AC Repair, and Cleaning. The application focuses on providing a simple, fast, and user-friendly experience with an attractive UI.

---


## ✨ Features

- 🏠 Modern landing page with gradient hero section
- 🔌 Popular home services with pricing
- 🛠️ Services include:
  - Electrician
  - Plumbing
  - AC Repair
  - Cleaning
- ⭐ Customer testimonials and ratings
- 📝 Service booking form
- 📍 Location selection
- 📱 Fully responsive design
- 🎨 Clean and attractive user interface
